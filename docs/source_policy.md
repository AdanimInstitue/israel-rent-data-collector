# Source Policy

This repository is limited to Category-1 public-safe sources.

## Active Source Registry

- `data_gov_il_locality_registry`

## Allowed Handling

- fetch the published locality registry through the public CKAN API;
- normalize locality codes, names, and district metadata;
- publish the resulting locality crosswalk with attribution and manifest metadata.

## Disallowed Handling

- Category-2 to Category-5 source adapters;
- derivative rent benchmark publication;
- references to non-public repositories, workflows, or storage locations.
