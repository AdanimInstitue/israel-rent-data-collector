# Output Model

## `locality_crosswalk.csv`

Columns:

- `locality_code`
- `locality_name_he`
- `locality_name_en`
- `district_he`
- `district_en`
- `population_approx`
- `source`

## `source_inventory.csv`

Registry export of the active public-safe sources used by this collector.

## `manifest.json`

Bundle manifest with:

- collector version
- generation timestamp
- source summary
- tracked file artifacts with relative paths, hashes, bytes, and row counts
